# Prompt Pack — HVHC BigData Science Domain (M20–M26)

Bộ prompt này đã được cập nhật theo file Excel:

- `HVHC_BigData_Thiet_ke_ky_thuat_chi_tiet_M20_M26.xlsx`

## Mục tiêu
Giúp Claude trong VS Code triển khai đúng các module nghiên cứu khoa học theo **mã mới M20–M26**, đồng thời tái sử dụng đúng các module nền:
- M01
- M02
- M13
- M18
- M19

## Cách dùng
1. Dán nội dung `00-system-prompt.md` vào đầu phiên làm việc.
2. Mở `INDEX.md` để chọn module.
3. Với mỗi module:
   - dùng phần “Prompt lập kế hoạch”
   - sau đó dùng phần “Prompt thực thi theo phase”
   - cuối cùng dùng phần “Prompt tự review”
