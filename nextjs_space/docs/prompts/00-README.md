# 00-README.md

Bộ này là **prompt pack siêu chặt theo từng sprint** cho Claude trong VS Code.

## Thứ tự dùng
1. Đưa `00-SYSTEM-SPRINT-GUARD.md` vào đầu phiên.
2. Chọn đúng sprint cần làm.
3. Không cho Claude làm quá 1 sprint trong 1 lượt lớn.
4. Sau mỗi sprint, bắt Claude self-review trước khi sang sprint sau.

## Trục sprint
- Sprint 01–03: nền domain khoa học
- Sprint 04–05: M20
- Sprint 06–09: M21, M23, M24, M25
- Sprint 10–11: M22, M26
- Sprint 12–13: hardening, review, release readiness
